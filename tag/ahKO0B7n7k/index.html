在这里简单介绍下双拼输入法，双拼输入法其实是键位的重新组合。通过自定义键盘而映射声母和韵母。比如当使用双拼输入法输入「双拼」时，依据自然码的方案： 按键U，对应声母U，韵母sh 按键D，对应声母D，韵母iang和uang 按键P，对应声母P，韵母un 按键N，对应声母N，韵母in 同时，有人将字形引入至输入法，组成了音码和形码结合的输入法，我们常说为辅码。以自然码的双拼方案为例：第一辅码是部首的声母，第二辅码是部首外的声母。比如辅这个字，fu为辅的双拼或全拼，然后将其拆成 车 和甫，那么它的辅码就是if，当我们输fuif时候选词诗就上屏显示了。 实际中仅打了第一个辅码windows 自带的输入法虽然支持双拼的，但是不支持辅码。搜狗等输入法需要通过tab键来开启辅码，暂时发现只有手心输入法是支持这种直接辅码模式的。 Markdown 工具 Markdown 是一种轻量级标记语言，对于笔者来说Markdown 的意义并不是在于取代word，而是让我们敲击键盘时专注于写作本身，忘记排版问题给我们的干扰。MarkDown 生成标题只需要输入# 敲击空格
